# Lipa na Mpesa Starter-Kit using Node.js
Steps for using Lipa na Mpesa Starter-Kit 

* Clone the repository.
* Run the following command:
`npm install`
Go through the `index.js` file.

* Provide your credentials on the `.env` file.

* Start the server:
`npm run dev`

* Start Ngrok
`npm run ngrok`
